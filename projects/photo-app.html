Photo App

Description

APR-JUN 2022

In my Intro to Web Development class, our class project was to develop a website using REST APIs and Heroku that allows users to login and view their friends' posts, with authencation and stored user functionality.

Made to resemble the popular social media app, Instagram, navigating to the website will ask the user to login. Once the user logs in, they will be presented a feed of photos from people they follow. Users can follow and unfollow users, like and bookmark posts, and make comments on posts.

Frameworks/Packages

HTML, CSS, JavaScript, Python (Flask), pSQL, Heroku
